CRAN 镜像使用帮助
======================

镜像地址
----------

`http://mirrors.ucas.ac.cn/cran/ <http://mirrors.ucas.ac.cn/cran/>`_


使用帮助
----------

1. 在文件 ``.Rprofile`` (Linux 下位于 ``~/.Rprofile``) 末尾添加

::

    options("repos" = c(CRAN="http://mirrors.ucas.ac.cn/cran/"))
